# Zpdf
Zpdf is a PDF document processing platform based on **fine-tuned large language models**, featuring professional-grade parsing and processing capabilities for special document types such as **complex PDF layouts**, **LaTeX mathematical formulas**, **chemical molecular formulas**, **image tables**, and **watermarked PDFs**. It is suitable for **scientific research**, **education**, and **office** scenarios.

## Product Features
The tool revolves around 4 core functions to provide users with a full-process PDF processing solution:

**1. High-Precision PDF to Markdown Parsing and Translation**
- Precisely identify and extract text, formulas, tables, and other content from PDFs, convert to editable Markdown format with one click, adapt to large model input requirements, and support multi-language translation simultaneously.

## Changelog
- **Zpdf-v1.2 - Released on 2025/12/03**

  **Update Content for PDF Translation with Preserved Layout:**
  - Integrated a new model to solve the problem of untranslated partial LaTex formulas in the old version.
  - Directly provide a two-column comparison output format, eliminating the need to switch back and forth between the original text and translated text.
  - Further optimized the original format preservation and translation output for PPT-type PDFs, improved the output adaptive function, and reduced output noise.
  
  **Update Content for PDF Parsing:**
  - Further optimized the parsing robustness for blurred documents and documents with complex layout formats.
  
  **Other Update Content:**
  - Fixed the repeated download issue caused by network environment.
  - Fixed other known issues.

- **Zpdf-v1.0.1 - Released on 2025/11/17**
  - The **PDF Parsing** function has integrated a **new model**, with comprehensively improved recognition accuracy and significantly increased translation efficiency.
  - The **Handwritten Image Recognition** function has integrated a **new model**, with comprehensively improved recognition accuracy.
  - The web version interface has been fully upgraded, providing a clearer and more intuitive visual and operational experience.
  - Fully supports **recognition, parsing, and translation** of Chinese, English, Japanese, Korean, French, German, Spanish, and Russian.
  - Integrated a historical **file management system**, with completed files retained for **7** days.
  - Integrated a **user management system**, supporting one-click login for users.
  - Fixed some known issues.
